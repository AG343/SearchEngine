Easy Set Up Steps

1. Get the New Tab Changer from the Chrome Web Store
2. Go to Chrome Settings(chrome://settings)
3. Click "On Startup" on the left side of the Screen
4. Click "Open a specific page or set of pages"
5. Click Add a new page
6. Enter the URL from the homepage and click Save
7. Done!
